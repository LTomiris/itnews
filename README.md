# 📰 IT News Portal

Учебный проект — новостной портал на тему IT и технологий на казахском языке.

## Стек
- HTML5
- CSS3

## Функциональность
- Лента новостей с изображениями и описаниями
- Категории: AI, Java, JavaScript, Cloud, Серверы
- Сайдбар с популярными темами
- Навигация по разделам
- Адаптивная сетка статей
- Подвал с копирайтом

## Структура файлов
- `index.html` — главная страница
- `style.css` — стили сайта
